# User Guide

## Auto Apply

The Auto Apply feature automates the process of finding and applying to jobs.

### Getting Started

1.  **Configure**: Go to the "Auto Apply" page.
2.  **Set Keywords**: Enter keywords like "Python", "Remote".
3.  **Set Locations**: Enter locations like "United States".
4.  **Set Daily Limit**: Choose how many jobs to apply to per day.
5.  **Save**: Click "Save Configuration".
6.  **Start**: Toggle the "Auto Apply" switch to ON.

### Monitoring

-   **Activity Log**: View the status of daily cycles in the "Activity" tab.
-   **Queue**: View jobs that require manual review (e.g. external sites) in the "Queue" tab (coming soon).

### Features

-   **Rate Limiting**: Automatically respects platform limits (e.g. 5/hr on LinkedIn).
-   **Duplicate Detection**: Skips jobs you've already applied to.
-   **Resume Selection**: Uses your default resume.
-   **Cover Letter**: Can generate AI cover letters (if configured).

### Troubleshooting

-   **No jobs applied?**: Check if your daily limit is reached or if no jobs match your keywords.
-   **Errors?**: Check the "Errors" column in the activity log.
